About CoachZilla

CoachZilla is a transit planning application designed to help users locate bus stops, view station information, and build efficient travel routes.

The system integrates map-based navigation with station and route data to provide a streamlined travel planning experience.

How to Use

  • Login or create an account to access the system
  • Use the main page map to view stations and routes
  • Click on stations to view detailed information
  • Navigate to "Search Routes" to find stops
  • Add stops to your travel plan
  • View and manage your plan in the View Travel Plan page
